# Список задач 

Простое веб-приложение «Список задач», созданное на Vue 3 с использованием Composition API и `<script setup>`.

## Возможности
Добавление новых задач
Отметка задач как выполненных (чекбокс)
Редактирование текста задачи (двойной клик по тексту)
Удаление задач
Фильтрация: Все, активные, выполненные
Автоматическое сохранение задач в localStorage
Плавные анимации при добавлении, удалении и перемещении задач
Кнопка «+ Add» активна только при заполненном поле

## Технологии
Vue 3 (Composition API + `<script setup>`)
Vite
localStorage для хранения данных
Чистый CSS (без дополнительных библиотек)

## Как запустить проект

1. Нужен **Node.js** (Версии незнаю выше 10+)
   - Качать тут: https://nodejs.org (Советую LTS-версия)

2. Открываеи командную строку в папке проекта

3. Качаем зависимости: npm install 
 4. Запускаем в режиме дев
    npm run dev
5. Радуемся на  http://localhost:5173
6. Вы велеколепны раз небыло ошибок
